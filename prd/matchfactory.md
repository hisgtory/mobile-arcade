# 매치 팩토리 (Match Factory)

> 물건을 매칭하여 공장 주문을 완수하는 매치-3 퍼즐

## 개요

매치 팩토리는 스왑 방식 매치-3 퍼즐에 **공장 주문(Order)** 메타를 결합한 게임이다.
플레이어는 보드의 타일을 스왑하여 3개 이상의 같은 재료를 매칭하고,
매칭된 재료는 공장 주문(Order) 달성에 카운트된다.
제한된 무브 내에 모든 주문을 완수하면 스테이지 클리어.

## 장르 위치

| 게임 | 코어 메카닉 | 메타 | 차별점 |
|------|-------------|------|--------|
| 로얄매치(#11) | 스왑 매치-3 | 왕국 꾸미기 | 장식 메타 |
| 크런치3(crunch3) | 스왑 매치-3 | 점수 목표 | 단순 스코어 |

## 게임 규칙

### 기본 규칙
- 8×8 보드에 다양한 재료 타일이 배치됨
- 인접 타일을 스왑하여 가로/세로 3개 이상 매칭
- 매칭된 타일의 종류와 수량이 주문(Order) 달성에 카운트
- 제한된 무브 내에 모든 주문 달성 시 **스테이지 클리어**
- 무브를 모두 소진해도 주문 미달성 시 **게임 오버**

### 주문(Order) 시스템
- 각 스테이지에 2~4개의 주문이 존재
- 주문 = "특정 재료를 N개 수집" (예: 🍎 ×10, 🍊 ×8)
- 매칭 시 해당 재료가 주문 카운트에 반영
- 모든 주문의 카운트가 목표에 도달하면 클리어

### 콤보 & 캐스케이드
- 매칭 후 빈 자리에 위 타일이 떨어짐 (gravity)
- 떨어진 후 다시 매칭 가능 → 캐스케이드
- 캐스케이드로 발생한 매칭도 주문에 카운트됨

## 로얄매치 대비 장점

1. **명확한 목표**: 점수가 아닌 구체적 수집 목표 → 플레이어가 전략적 스왑
2. **공장 테마**: 재료 수집 → 주문 완수 → 생산의 만족감
3. **확장성**: 주문 복잡도, 특수 재료, 공장 업그레이드 등 확장 가능
4. **빠른 개발**: crunch3 엔진 패턴 재사용, 주문 시스템만 추가
