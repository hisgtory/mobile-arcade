# 워드퍼즐 (Word Puzzle) PRD

## 개요
한국어 단어 퍼즐 게임. 격자에 숨겨진 한국어 단어를 찾아 선택하는 게임.

## 게임 메커니즘
- **격자 기반**: 5×5 ~ 6×6 격자에 한국어 글자가 배치
- **단어 찾기**: 가로/세로로 연결된 셀을 탭하여 단어 완성
- **자동 감지**: 올바른 단어 선택 시 자동으로 인식
- **힌트**: 첫 번째 미발견 단어의 위치 표시
- **스테이지**: 30개 퍼즐 (Easy 3단어 / Medium 4단어 / Hard 5단어)

## 스테이지 구성
| Stage | 난이도 | 격자 | 단어 수 |
|-------|--------|------|---------|
| 1-10  | Easy   | 5×5  | 3       |
| 11-20 | Medium | 5×5  | 3-4     |
| 21-30 | Hard   | 6×6  | 5       |

## 점수 체계
- 단어 발견: 글자 수 × 100점
- 스테이지 클리어: 모든 단어 발견

## 기술 스택
- **lib/wordpuzzle**: Phaser.io 기반 게임 코어
- **web/arcade**: React + Stitches UI
- **rn**: WebView 래핑

## UI 구성
- HUD: 스테이지, 발견 단어 수/전체, 점수, 힌트/재시작 버튼
- 게임 화면: 격자 + 하단 단어 목록
- 클리어 화면: 결과 표시 + 다음 스테이지
