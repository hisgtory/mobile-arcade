# 포레스트 애니팝 (Forest Anipop) — PRD

> Issue: #110 | Genre: match-3 (tap-to-pop) | Rating: 4.3 | Developer: springpocket

## 1. 개요

숲 테마의 매치 퍼즐. 같은 종류의 타일이 3개 이상 연결된 그룹을 탭하면 팝(제거)되는 캐주얼 매치 퍼즐.

## 2. 코어 메카닉

### 2.1 탭-투-팝 (Tap-to-Pop)
- 기존 crunch3의 "스와이프 매치"와 차별화
- **같은 종류 타일이 3개 이상 상하좌우로 연결**된 그룹을 탭하면 한 번에 제거
- 제거 후 중력 적용 → 빈 자리에 새 타일 생성 → 연쇄(cascade) 가능
- 큰 그룹을 한 번에 터뜨릴수록 높은 점수 (전략적 기다림 유도)

### 2.2 스코어링
- 3개: 100점
- 4개: 200점
- 5개+: 500점
- 콤보 배수 적용 (연쇄 시 x2, x3...)

### 2.3 스테이지 시스템
- 제한된 moves 내에 목표 점수 달성
- 5개 스테이지 (MVP)
- 난이도: 타일 종류 수 증가 + moves 감소 + 목표 점수 증가

## 3. 숲 테마

### 3.1 타일: 자연/숲 느낌의 과일 & 채소
- fruit_apple, fruit_cherry, fruit_blueberry, fruit_peach, fruit_kiwi
- fruit_strawberry, fruit_orange, fruit_grape_red
- vegetable_carrot, vegetable_corn

### 3.2 비주얼
- 배경: 숲 녹색 (#E8F5E9)
- 타일 배경: 밝은 녹색 테두리
- 전반적으로 자연 친화적 색감

## 5. 매치-3 장르 분석

- 매치-3은 포화 장르지만, **탭-투-팝은 스와이프와 다른 경험**
- 숲 테마 + 귀여운 픽셀아트 → 차별화 포인트
- MVP 1주 개발 → CPI 테스트 → 데이터로 판단
