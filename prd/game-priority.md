## 판단 기준

1. **구현 난이도**: 낮을수록 우선
2. **시장 검증**: 앱스토어 상위 장르 우선
3. **에셋 요구**: 최소 에셋(숫자, 색상, 기본 도형)으로 가능한 것 우선
4. **공통 인프라 재사용**: lib/web/rn 파이프라인 + 브릿지 그대로 사용
5. **IP 리스크**: 없어야 함
